The ``1-my_list`` module
========================

Using ``MyList``
----------------

Import the class:
    >>> MyList = __import__('1-my_list').MyList

Create an instance and add elements:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test print_sorted:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Check if the original list is still unsorted:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test with an empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Test with already sorted list:
    >>> sorted_list = MyList([1, 2, 3])
    >>> sorted_list.print_sorted()
    [1, 2, 3]

Test with negative numbers:
    >>> neg_list = MyList([1, -5, 10, 0])
    >>> neg_list.print_sorted()
    [-5, 0, 1, 10]
